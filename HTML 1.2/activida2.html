Etiquetas y atributos necesarios para insertar imágenes:

  • img: etiqueta para insertar una imagen.
  • src: especifica la ruta de la imagen (relativa o absoluta).
  • alt: texto alternativo para accesibilidad.
  • title: texto que aparece al pasar el cursor (opcional).
  • width y height: definen tamaño de la imagen.
  • style: permite aplicar estilos como bordes o márgenes (opcional).
  • a: convierte una imagen en enlace.
  • href: URL de destino cuando se hace clic.
  • target="_blank": abre el enlace en otra pestaña (opcional).
